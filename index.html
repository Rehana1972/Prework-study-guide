Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is one of the three cornerstone languages used in web development.
  • HTML is used to build the structure of a page.
  • HTML is used in navigating the internet.
  • HTML is used as a storage function in the browser.
  • HTML is used for data entry support.
  • HTML is used for game development.
  • HTML is used for offline web applications.
  • HTML stands for Hypertext Markup Language.

CCS

  • CSS stands for Cascading Style Sheets.
  • CSS defines the colours, aesthetics, visual layout and organization on a page.
  • CSS is used to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • CSS is also used to add animations and other decorative features.

Git

  • Git is a version control system that tracks changes in any set of computer files.
  • Git is used for coordinating work among programmers collaboratively developing source code during software development.
    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia and animate images.
  • JavaScript is a high-level dynamic programming language that allows users to interact with websites.
  • Using JavaScript allows a developer to power their websites with effects, interactivity, data communication and much more.
